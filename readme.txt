== Description ==
A customizable login widget that will allow you to add a welcome message, the users avatar, custom links, and cubepoints integration.
A few features:<br />
* Add welcome message <br />
* Add custom links.<br />
* Add userlevel specific links.<br />
* Show user avatar.<br />
* Style every aspect using CSS
* And there is so much more to come in future releases.<br />

== Installation ==
1. Upload the 'custom-login-with-cube-points-integration' folder to the `/wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place the Custom Login Widget on your sidebar!
4. Configure the custom login settings using the graphical interface.
